## 1. 介绍
#### Whishow是实时播放帧级别音频流和视频流的工具，支持rtsp/rtmp/mp4等输入

![avatar](img/img2.png)

## 2. 使用

./whishow.exe <video path>

  例如：
  
  离线文件 ./whishow.exe test.mp4
  
  在线流 ./whishow.exe rtmp://mobliestream.c3tv.com:554/live/goodtv.sdp
